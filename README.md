# PromptCraft AI ⚡

Welcome to Day 16 of the **20Day20AI Projects Challenge**! `PromptCraft AI` is a clean, cyber-inspired frontend tool designed to transform raw, simple ideas into highly optimized, structured, and professional AI prompts.

## 🌟 Features
- **Dynamic Optimization:** Choose custom variables like Tone, Target Audience, and Length to shape your prompt.
- **Instant Templates:** One-click presets for common use-cases (Copywriting, Coding, Brainstorming).
- **Interactive UI:** A highly responsive cyber-vibe glassmorphic interface.
- **One-Click Copy:** Seamlessly copy your engineered prompt to your clipboard instantly.

## 🛠️ Tech Stack
- HTML5 (Semantic Structure)
- CSS3 (Custom Properties, Flexbox/Grid, Glassmorphism)
- JavaScript (DOM Manipulation, Dynamic String Interpolation)
